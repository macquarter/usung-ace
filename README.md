# 유성에이스 (USUNG ACE)

후드·덕트 전문 제작/시공 14년 — 유성에이스 공식 웹사이트 & CMS.

## 구성

- `index_v6.html` — 메인 웹사이트 (최신 버전)
- `admin.html` — CMS 관리자 페이지 (7섹션: 대시보드 / 콘텐츠 / 문의 / 갤러리 / 게시판 / 챗봇 / 설정)
- `chatbot.js` — "에이스봇" 전문 CS 챗봇 (룰 기반, 50+ 블로그 포스트 학습, 전화번호 자동 문의 등록)
- `data.json` — 콘텐츠 데이터

## 챗봇 기능

- 유성에이스 네이버 블로그 50개 포스트 학습
- 한국어 자연어 인식 (창업/오픈/견적/AS/출장/가격 등)
- 전화번호 자동 감지 → 관리자 문의내역에 즉시 등록
- 한글 IME 조합 버그 해결
- Pretendard 웹폰트 적용

## 관리자 페이지

localStorage 기반 CMS — 필드 편집 / 라이브 편집 모드 지원.
챗봇 지식베이스(KB) 편집, 대화 로그, 챗봇 설정 관리 포함.
